## 关于 iV2EX

iV2EX 是一个现代化的 V2EX 客户端，使用 Next.js 构建，提供更好的阅读体验和深色模式支持。

## 技术栈

- **前端框架**：Next.js + React
- **UI 组件**：现代化的 UI 设计
- **部署平台**：Vercel
- **特性**：深色模式支持、响应式设计

## 项目架构

iV2EX 采用 Next.js 作为核心框架，充分利用了 Next.js 的服务端渲染（SSR）和静态生成（SSG）能力，为用户提供快速、流畅的浏览体验。

## 主要功能

- **现代化界面**：简洁美观的用户界面设计
- **深色模式**：支持深色模式，提供更好的夜间阅读体验
- **响应式设计**：适配各种设备屏幕尺寸
- **快速加载**：优化的性能，提供流畅的浏览体验
- **搜索功能**：便捷的内容搜索
